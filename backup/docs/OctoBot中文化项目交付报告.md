## 🔧 技术支持

### 常见问题处理

#### 问题1：翻译未生效

**解决方案**：
1. 清除浏览器缓存（Ctrl + Shift + Delete）
2. 硬刷新页面（Ctrl + Shift + R）
3. 检查translator.js是否正确加载
4. 查看浏览器控制台是否有错误

#### 问题2：登录失败

**解决方案**：
1. 检查用户名和密码是否正确
2. 检查认证服务器是否运行
3. 查看认证服务器日志：`tail -f /root/octobot-custom/auth/auth.log`
4. 重启认证服务器

#### 问题3：OctoBot无法访问

**解决方案**：
1. 检查OctoBot容器是否运行：`docker ps`
2. 检查Nginx是否运行：`systemctl status nginx`
3. 查看Nginx日志：`tail -f /var/log/nginx/error.log`
4. 重启相关服务
