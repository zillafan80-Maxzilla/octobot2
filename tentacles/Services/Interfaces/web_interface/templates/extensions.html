使用指南

  1. 点击上方"立即使用策略设计器"按钮
  2. 选择您想要配置的策略类型（Grid Trading或DCA Trading）
  3. 填写策略参数
  4. 点击"验证配置"查看分析结果
  5. 点击"保存到OctoBot"应用配置
  1. 访问"配置文件"页面（点击这里）
  2. 选择您的策略配置文件
  3. 点击"配置"按钮
  4. 修改参数后保存并重启OctoBot
提示：所有15个预置策略配置文件已解锁，您可以自由编辑！

配置修改后需要重启OctoBot才能生效：

  1. 保存配置后，系统会提示您重启
  2. 点击"保存并重启"按钮
  3. 等待OctoBot重新启动（约30-60秒）
  4. 重启完成后，新配置即生效
