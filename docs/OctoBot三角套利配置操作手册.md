## 系统概述

### 什么是三角套利？

三角套利（Arbitrage Trading）是一种利用不同交易所或同一交易所内不同交易对之间的价格差异来获利的交易策略。

**工作原理：**
1. 监控多个交易所的交易对价格
2. 计算跨交易所的平均价格
3. 当某个交易所的价格与平均价格偏离足够大时，触发套利交易
4. 在本地交易所以当前价格买入/卖出
5. 在价格回归平均值时平仓获利

**优势：**
- ✅ 低风险：利用价格滞后，不依赖市场方向
- ✅ 自动化：机器人24/7监控和执行
- ✅ 多交易所：可以同时监控多个交易所

## 三角套利策略配置

### 当前配置

OctoBot已预配置"Arbitrage 交易"策略，无需额外配置。

### 查看策略配置

1. 点击顶部导航栏的"配置文件"菜单
2. 查看当前配置：**Arbitrage 交易**
3. 点击"策略"标签查看详细配置

### 策略说明

**套利交易模式特点：**
- 监控所有配置的交易所价格
- 计算跨交易所平均价格
- 当价格偏离平均值时触发交易
- 创建2个订单：
  1. 限价买入/卖出订单（当前本地价格）
  2. 平仓订单（平均价格）或止损订单

**推荐配置：**
- 启用2-3个交易所用于套利交易
- 添加更多高流动性交易所作为价格参考
- 价格参考交易所不需要API密钥

## 模拟盘配置

### 为什么使用模拟盘？

- ✅ 零风险测试策略
- ✅ 验证配置正确性
- ✅ 学习系统操作
- ✅ 优化参数设置

### 配置步骤

#### 步骤1：进入交易配置

1. 点击"配置文件"菜单
2. 点击"交易"标签

#### 步骤2：配置交易设置

**参考市场（Reference Market）：**
- 默认值：USDT
- 说明：用于计算盈利和投资组合总价值的参考货币

**风险（Risk）：**
- 默认值：0.5
- 范围：0-1
- 说明：
  - 0 = 非常保守，小额交易
  - 0.5 = 平衡
  - 1 = 非常激进，大额交易

#### 步骤3：启用模拟交易

**Trader（真实交易）：**
- ⚪ **Enabled：未勾选**（重要！）
- ⚪ Load-Trade-History：未勾选

**Trader simulator（模拟交易）：**
- ✅ **Enabled：已勾选**（必须！）

#### 步骤4：配置初始资金

**手续费设置：**
- Maker: 0.1%
- Taker: 0.1%

**初始投资组合（Starting Portfolio）：**

**方案A：1000美元全USDT**
```
BTC: 0
USDT: 1000
```

**方案B：1000美元混合配置**
```
BTC: 0.01 (约916美元)
USDT: 84
```

#### 步骤5：保存并重启

1. 点击"APPLY CHANGES AND RESTART"按钮
2. 等待60秒让系统重启
3. 刷新页面

#### 步骤6：验证配置

1. 点击"投资组合"菜单
2. 查看"投资组合总值"
3. 确认初始资金正确显示

## 常见问题

### Q1: 为什么没有看到任何交易？

**可能原因：**
1. 市场价格没有足够的套利机会
2. 风险参数设置过低
3. 只配置了一个交易所（需要多个交易所才能套利）

**解决方案：**
- 添加更多交易所作为价格参考
- 适当提高风险参数（0.5-0.7）
- 耐心等待市场出现套利机会

### Q2: 如何添加更多交易所？

1. 进入"账户"菜单
2. 点击"添加交易所"
3. 选择交易所并配置API密钥
4. 保存并重启

**推荐交易所：**
- Binance（已配置）
- OKX
- Huobi
- Kraken
- Coinbase Pro

### Q3: 模拟盘和实盘有什么区别？

**模拟盘：**
- ✅ 使用虚拟资金
- ✅ 零风险
- ✅ 不需要真实API密钥（仅用于价格数据）
- ❌ 无法提现

**实盘：**
- ⚠️ 使用真实资金
- ⚠️ 有风险
- ✅ 需要有效的API密钥和权限
- ✅ 可以提现

### Q4: 如何切换到实盘交易？

**⚠️ 警告：实盘交易有风险，请确保充分测试后再切换！**

1. 进入"配置文件" → "交易"
2. 启用"Trader" → "Enabled"
3. 禁用"Trader simulator" → "Enabled"
4. 保存并重启
5. 确认顶部显示"实盘交易"

### Q5: 配置保存后没有生效怎么办？

**解决步骤：**
1. 确认点击了"APPLY CHANGES AND RESTART"
2. 等待至少60秒让系统完全重启
3. 强制刷新浏览器（Ctrl+Shift+R）
4. 重新检查配置

### Q6: 如何修改初始资金？

1. 进入"配置文件" → "交易"
2. 找到"Trader simulator" → "Starting Portfolio"
3. 修改BTC和USDT数量
4. 点击"APPLY CHANGES AND RESTART"
5. **注意：** 修改后会重置历史数据

### Q7: 手续费如何设置？

**Binance现货手续费：**
- Maker: 0.1%
- Taker: 0.1%
- VIP用户可能更低

**配置位置：**
- "配置文件" → "交易" → "Trader simulator" → "Fees"

### Q8: 如何导出交易记录？

1. 进入"交易"菜单
2. 点击"交易记录"标签
3. 点击"导出"按钮
4. 选择CSV或JSON格式

### Q9: 系统出现502错误怎么办？

**可能原因：**
- OctoBot服务崩溃
- Nginx代理超时

**解决方案：**
1. 等待几分钟自动恢复
2. 联系系统管理员
3. 检查服务器日志

### Q10: 如何优化套利策略？

**优化建议：**
1. **增加交易所数量**：更多价格参考
2. **调整风险参数**：根据市场波动性
3. **选择高流动性交易对**：BTC/USDT, ETH/USDT
4. **监控手续费**：确保套利利润大于手续费
5. **设置合理的止损**：保护资金
